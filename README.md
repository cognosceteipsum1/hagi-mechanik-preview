# Hagi Mechanik – Konzeptvorschau

Diese statische, responsive Vorschau zeigt eine mögliche moderne Startseite für die Hagi Mechanik AG. Sie ist ausdrücklich als unverbindliche Konzeptstudie gekennzeichnet und für Suchmaschinen gesperrt.

## Vorschau öffnen

Am einfachsten `index.html` doppelklicken. Für eine realistischere lokale Vorschau im Projektordner einen kleinen Webserver starten und danach `http://localhost:8000` öffnen.

```sh
python3 -m http.server 8000
```

## Enthalten

- moderne, mobil optimierte Startseite
- echte Bilder und belegte Inhalte der bestehenden Firmenwebsite
- klare Leistungsstruktur und Offert-CTA
- lokale Signale für Wünnewil / Schweiz
- FAQ für Suchmaschinen und KI-Systeme
- strukturierte Organisationsdaten
- `noindex`- und `robots.txt`-Schutz für die private Vorschau

Vor einer Veröffentlichung müssen Texte, Zertifikate, Produktionsmöglichkeiten, Datenschutz/Impressum und alle Kontaktdaten mit Hagi Mechanik bestätigt werden.
